# kok
机场管理xray面板，实现后台管理，以及多用户登录，可满足用户自订阅等。
# 功能介绍

- [ ] 支持多用户注册，网页可视化操作
- [ ] 前后端分离架构，支持前端页面样式丰富
- [ ] 全新设计的用户界面，使用vue3+element-plus
- [ ] 支持一键订阅
- [ ] 支持后台节点管理
- [x] 接入mysql数据库，实现数据持久化
